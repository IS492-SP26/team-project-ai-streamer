# Five-Minute Interview Guide

Opening: “Thanks for helping. This is a class project prototype about AI livestream safety. We are testing whether the risk panel and AI responses make sense to a human evaluator. We are not collecting sensitive personal information. Your feedback is voluntary and anonymous. You can stop anytime.”

During tasks: “Please think aloud if something is confusing. I may time each task, but this is not testing you; it is testing the system.”

Questions:
1. In your own words, what did the risk state/action mean?
2. Which response felt most useful?
3. Which response felt confusing, too strict, or too weak?
4. Did the AI stay enough like a livestream persona?
5. Would you trust this as a support tool for a small creator/moderator? What would you need before trusting it more?

Closing: “Please complete this short survey. The most useful feedback is what felt confusing, untrustworthy, too aggressive, or too weak.”
